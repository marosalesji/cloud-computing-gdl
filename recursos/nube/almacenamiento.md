# Almacenamiento en la nube

Hay 3 tipos principales de almacenamiento:

- File based storage
    - AWS EFS
- Block based storage
    - AWS EBS
- Object based storage
    - AWS S3

## AWS Simple Storage Service (S3)

Es un servicio de almacenamiento de objetos que ofrece escalabilidad, alta durabilidad y seguridad para almacenar y recuperar
cualquier cantidad de datos desde cualquier lugar en la web.
