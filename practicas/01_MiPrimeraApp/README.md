# Práctica 1: Mi primera app en la nube

El objetivo de esta práctica es implementar la aplicación que diseñaste en la Tarea 2,
utilizando EC2 para cómputo y S3 para almacenamiento.

## Requisitos

- EC2 para el cómputo
- S3 para almacenamiento
- Implementación en Python o TypeScript

No se espera una aplicación compleja, pero sí funcional y coherente con la arquitectura
que propusiste. Puedes ajustar el alcance si es necesario, siempre que mantengas la idea
central y el uso de ambos servicios.

## Entregables

### Reporte en PDF

- Portada con nombre del alumno, nombre de la actividad, materia y fecha
- Descripción del proyecto
  - Nombre de la aplicación
  - Problema que busca resolver
- Diagrama de arquitectura con los componentes de AWS y tu programa, mostrando cómo interactúan
- Descripción del diagrama (200 a 300 palabras) explicando cómo funciona la arquitectura y cómo se integran los servicios
- Link al repositorio de GitHub
- Link al video
- Referencias si las hay

### Repositorio de GitHub

- Código del proyecto — claro, entendible y funcional
- README con explicación breve del repositorio

### Video (5 a 10 minutos)

El video debe mostrar la aplicación corriendo en la nube, no en local. Debe incluir:

- La instancia de EC2 encendida en la consola de AWS
- El bucket de S3 con archivos reales
- Explicación breve de qué hace la aplicación y cómo utiliza EC2 con S3
